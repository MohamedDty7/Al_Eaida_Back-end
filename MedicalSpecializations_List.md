## كيفية الاستخدام

يمكن الآن استخدام هذه التخصصات في:

1. **إنشاء الأطباء** - اختيار التخصص المناسب لكل طبيب
2. **البحث والفلترة** - البحث عن الأطباء حسب التخصص
3. **إدارة المواعيد** - ربط المواعيد بالتخصصات
4. **التقارير** - إحصائيات حسب التخصصات

## API Endpoints المتاحة

- `GET /api/Specialization` - الحصول على جميع التخصصات
- `GET /api/Specialization/{id}` - الحصول على تخصص محدد
- `POST /api/Specialization` - إضافة تخصص جديد
- `PUT /api/Specialization` - تحديث تخصص موجود
- `DELETE /api/Specialization/{id}` - حذف تخصص

## ملاحظات مهمة

- جميع التخصصات نشطة (`IsActive = true`)
- تم إضافة تاريخ الإنشاء تلقائياً
- يمكن إضافة المزيد من التخصصات حسب الحاجة
- التخصصات مرتبة أبجدياً في قاعدة البيانات
